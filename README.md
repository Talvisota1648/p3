# p3
Создать небольшой CLI или веб-приложение, которое взаимодействует с API Nikta Graph и выполняет задачу:  Пользователь вводит короткое описание задачи в свободной форме (например: “нужно проверить прогресс команды за неделю”).  Приложение отправляет этот текст ИИ-агенту Nikta Graph через requests.post к /run_workflow_as_server или другому endpoint
